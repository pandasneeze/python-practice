# UNIT 7 출력 방법 알아보기
콤마로 구분 -> 공백으로 띄워서 한 줄로 출력
- print(값1, 값2, 값3)

## sep
값 사이에 문자열 삽입
- print(값1, 값2, 값3, sep = ', ') -> 값1, 값2, 값3
- print(값1, 값2, 값3, sep = '') -> 값1값2값3

## 줄바꿈
- print(1, 2, 3, sep = '\n') 
- print(1\n2\n3)
- print 함수는 기본적으로 값 끝에 줄바꿈을 포함
- \문자 자체를 출력할 때는 두 번 입력(\\)

## end
- print(1, 2, 3, end = ' ') -> 값 끝에 줄바꿈을 없애고 공백 삽입
